🔗 Integrasi Google Sheets (Realtime)

Kirim data otomatis ke Google Sheets via Google Apps Script webhook.

📋 Cara Setup:

  1. Buka Google Sheets → Extensions → Apps Script
  2. Buat fungsi doPost(e) yang menerima data JSON
  3. Deploy as Web App → Anyone can access
  4. Copy URL webhook di bawah ini
Belum ada URL

✅ Setelah URL disimpan, data kunjungan baru akan otomatis terkirim ke Sheets (realtime).

📋 Contoh Apps Script (doPost):
function doPost(e){var data=JSON.parse(e.postData.contents);var sheet=SpreadsheetApp.getActiveSheet();if(data.type==='realtime_push')sheet.appendRow([data.data.tgl_kunjungan,data.data.no_antrean,data.data.nama_pengunjung,new Date()])}
